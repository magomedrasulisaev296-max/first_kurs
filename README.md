# проект PythonProject1

Этот проект является mvp бекэнд кодом для страницы банка


## Описание:

Проект создан как практика по написанию подобного типа кодов.

Проект может выводить данные главной страницы банкаЖ

Топ 5 транзакций,
все карты,
а также курсы валют в реальном времени


## Реализованные модули и функции


### Модуль reports:

декоратор decorator
    
    """записывает вывод функции в json файл filename: авто имя или заданное пользователем"""

функция spending_by_category

    """выводит траты по категории и заданной дате на три месяца назад"""

### Модуль services.py

функция analyze_cashback_categories

    """возврощает ответ в формате Json по выбранному времени и категории из заданного списка библиотек"""

### Модуль utils.py

Функция greetings

    """выводит приветсвенное сообщение в зависимости от времени пользователя"""

Функция read_excel_file

    """возврощает excel файл"""

Функция all_cards 

    """выводит все номера карт имеющиеся в дата-фрейме(operations)"""

Функция top_transactions

    """выводит самые большие транзакции в переданном дата-фрейме"""

Функция currency_of_valuets

    """выдает курс валют к рублю в настоящие время"""

Функция currency_stoks

    """выдает курс заданных акций в настоящие время"""

### Модуль views.py

Функция veb_json

    """собирает функции из модуля utils.py и возвращает их в виде еденного json ответ"""
